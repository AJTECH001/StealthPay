sequenceDiagram actor EE as Employee participant SC as StreamCounter participant W as Aleo Wallet participant NET as Aleo Network SC->>SC: calculate() every 5s\nelapsedBlocks = elapsedMs / 5000\nclaimable = elapsedBlocks × streamRate EE->>SC: Click Claim Now SC->>W: requestRecords(stealthpay_payroll_v1.aleo) W-->>SC: [EmployeeRecord where payment_type == 1u8] SC->>W: executeTransaction(claim_stream_credits,\n[employee_record, claim_amount]) W->>W: Private phase:\nrecord_commitment = BHP256(salary ∥ secret)\ncompute accrued = elapsed_blocks × record_rate W->>NET: Broadcast transaction note over NET: finalize:\nverify record_commitment == salary_commitment\ncompute accrued_amount = elapsed_blocks × record_rate\nverify claim_amount <= accrued\ndeduct company balance\ntransfer_public_to_private → employee\nmint new EmployeeRecord (claimed_amount updated) NET-->>SC: tx_id SC-->>EE: ALEO/USDCX in wallet

Security Layers Stack

flowchart TB subgraph L1["Layer 1 — Zero-Knowledge Proofs"] ZK1[Salary amount never exposed on-chain] ZK2[Employee identity private in record] end subgraph L2["Layer 2 — Salary Commitment Scheme"] CS1[salary_commitment = BHP256 hash of salary + secret] CS2[Random salary_secret prevents brute-force] CS3[Claim verified by re-hashing — ZK proven equality] end subgraph L3["Layer 3 — Private Records"] PR1[EmployeeRecord encrypted to employee view key] PR2[Only owner can use it as transition input] end subgraph L4["Layer 4 — Access Control"] AC1[add_employee: only self.caller == employer] AC2[remove_employee: only self.caller == employer] AC3[claim: only record.owner can submit] AC4[withdraw: only self.caller == employer] end subgraph L5["Layer 5 — Balance Guards"] BG1[claim checks company_credits >= amount] BG2[claim checks accrued >= claim_amount for stream] BG3[withdraw checks company_credits >= amount] end subgraph L6["Layer 6 — Key Custody"] KC1[App never handles private keys] KC2[Wallet extension manages all signing] KC3[salary_secret generated client-side via crypto.getRandomValues] end L1 --> L2 --> L3 --> L4 --> L5 --> L6
